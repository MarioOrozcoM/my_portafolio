Bienes Raíces

Desarrollado con HTML, CSS, JS, PHP y MySQL

Descripción del Proyecto

Este sitio web está desarrollado pensado en un administrador de bienes raíces, el cuál cuenta con un solo usuario que sería el admin. Permite administrar el sitio, con la información relevante, administrar las propiedades que se ofertan. Los visitantes del sitio web pueden observar la información, revisar las propiedades ofertadas y ponerse en contacto con el administrador. Está desarrollado utilizando PHP y MySQL como backend, HTML, CSS y JS para el front.

Funcionalidades principales incluyen:

  • Administración del sitio web
  • Creación, edición, visualización y eliminación de las propiedades a ofertar
  • Modo Oscuro
  • Los clientes pueden comunicarse con el administrador

Tecnologías Utilizadas

HTML CSS JS PHP MySQL
